## 为什么选择 Paimon？ {#why-paimon}

<div className="feature-grid">
<div className="feature-item">

### 大规模分析 {#large-scale-analytics}
PB 级表，支持时间旅行、快速扫描规划、schema 演进与增量聚簇。

</div>
<div className="feature-item">

### 实时流式 {#realtime-streaming}
基于 LSM 的流式更新，支持多种合并引擎（去重、部分更新、聚合）与 changelog producer。

</div>
<div className="feature-item">

### 多模态 AI {#multimodal-ai}
在 Blob 表中存储图像、视频与 embedding。向量检索（DiskANN）、全文检索，以及面向 ML 管道的原生 Python SDK。

</div>
</div>
